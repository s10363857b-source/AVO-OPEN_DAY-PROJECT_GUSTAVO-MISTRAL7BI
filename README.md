Proposta alternativa per LLM del progetto GUSTAVO da presentare ai open-day
È stato scelto l'utilizzo di un LLM oper-source, in questo caso si tratta di Mistral 7B Instruct per risolvere questioni riguardanti sull'intelligenza e raccolta dati.
In questa proposta non si utilizzeranno delle informazioni inserite in prompt o da file .json ma bensì direttamente da file contenenti gli argomenti interessati (es. orari, regolamento d'istituto,ecc...) oppure
anche i file html delle pagine web.
Tale proposta è ancora da discuterne in classe per eventuali problematiche relative alle risorse hardware poiché Mistral 7B Instruct ne richiede una maggiore quantità rispetto alla proposta originaria.
In tal caso si penserà di adottarne una versione leggera (es. TinyLlama, Mistra small).
